# laptop-sales-analysis
Exploratory data analysis and sales insights from laptop sales data using Python and Power BI

## Project Overview
This project analyzes laptop sales data to identify key trends related to pricing, brand performance, and customer preferences. The analysis focuses on transforming raw sales data into meaningful insights that can support business and marketing decisions.

The project demonstrates data cleaning, exploratory data analysis, and interactive visualization using Python and Power BI.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel

## Key Objectives
- Clean and preprocess laptop sales data
- Analyze price distribution across brands and categories
- Identify top-performing laptop brands
- Segment laptops based on price ranges
- Visualize sales trends and key performance indicators

## Key Insights
- Identified top-performing laptop brands based on sales volume
- Price segmentation highlights budget, mid-range, and premium categories
- Clear relationship observed between price range and customer preference
- Dashboards provide interactive filtering for deeper analysis
